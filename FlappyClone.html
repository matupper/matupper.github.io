About

This is my first project in the Godot Engine. While not my first time playing around with game dev, this was my first project that had a clear end goal in mind. While it isn't either complicated or original, I built this project from scratch, including all the assets, such as the bird, pipes, and background. In the process, I learned a lot about the Godot Engine, and how to build a game from the ground up.

What I learned

  • How to implement sound effects
  • How to build a game from scratch
  • How to create and animate sprites
  • How to create a simple game loop
